    README for PDL::Ngrams

ABSTRACT
    PDL::Ngrams - Basic N-Gram utilities for PDL

REQUIREMENTS
    *   PDL

        Tested version 2.4.3

    *   PDL::VectorValued

        Should be available from the same place you got this module.

DESCRIPTION
    PDL::Ngrams provides some basic utilities for creation and manipulation
    of N-Grams over PDLs.

BUILDING
    Build this module as you would any perl module, by doing something akin
    to the following:

     gzip -dc distname-XX.YY.tar.gz | tar -xof -
     cd distname-XX.YY/
     perl Makefile.PL
     make
     make test                                     # optional
     make install

    See perlmodinstall(1) for details.
